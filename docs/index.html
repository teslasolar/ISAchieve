Quick Example

Here's a simple example of using ISAchieve:

// Initialize the achievement system
const achiever = new ISAchieve({
    agentId: 'my-agent-001',
    trackingEnabled: true
});

// Track an achievement

    timestamp: Date.now(),
    metadata: { version: '1.0.0' }
});
