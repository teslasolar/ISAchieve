# Contributing to ISAchieve

Thank you for your interest in contributing to ISAchieve!

## How to Contribute

1. **Fork the Repository**
   - Click the 'Fork' button on GitHub

2. **Clone Your Fork**
   ```bash
   git clone https://github.com/YOUR_USERNAME/ISAchieve.git
   cd ISAchieve
   ```

3. **Create a Branch**
   ```bash
   git checkout -b feature/your-feature-name
   ```

4. **Make Your Changes**
   - Write clean, documented code
   - Follow the existing code style
   - Add tests for new features

5. **Test Your Changes**
   ```bash
   npm test
   npm run lint
   ```

6. **Commit Your Changes**
   ```bash
   git add .
   git commit -m "Add: your feature description"
   ```

7. **Push to Your Fork**
   ```bash
   git push origin feature/your-feature-name
   ```

8. **Create a Pull Request**
   - Go to the original repository
   - Click 'New Pull Request'
   - Select your branch

## Code Style

- Use ES6+ features
- Follow ESLint rules (run `npm run lint`)
- Write clear comments
- Use meaningful variable names

## Testing

- Add unit tests for new features
- Ensure all tests pass before submitting
- Maintain or improve code coverage

## Documentation

- Update README.md if needed
- Add JSDoc comments to functions
- Update docs/ if adding new features

## Questions?

Open an issue for discussion before making major changes.
